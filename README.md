# Bleep it Professional

## Overview
An app utilizing ai to translate any statement into a work friendly one.

## User Stories
As a user, I want the ability to...
  - sign up
  - sign in
  - sign out
  - input a phrase I want reworked
  - save statements
  - update statements
  - delete statements
  - view statement index
  - view statement details
